# PlantCare-AI 🌿

An intelligent plant disease detection and care system powered by AI.

Upload a photo of a plant leaf and get:
- 🔍 **Disease Detection** — Instant prediction using a trained ResNet18 model
- 🤖 **AI Care Guide** — Detailed treatment advice powered by Grok AI
- 🌍 **Multilingual Support** — Translate care instructions to your language
- 🔊 **Voice Assistance** — Listen to care instructions via text-to-speech

## Supported Plants
Apple, Corn (Maize), Pepper, Potato, Tomato — 23 disease classes total.

Avialable on =[Plant Disease-AI](https://plantcare-ai-production.up.railway.app/)
